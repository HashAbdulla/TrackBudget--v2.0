# TrackBudget (v2.0)

TrackBudget is a secure, full-stack personal financial tracking web application. Originally developed as a collaborative capstone project at the University of Missouri-Kansas City, this independent repository serves as my personal standalone version. 

I am actively refactoring the core architecture, optimizing database workflows, and implementing advanced security baselines to transition the application into a production-ready system.

---

## 🛠️ Tech Stack & Infrastructure
* **Frontend:** React, HTML5, CSS3, JavaScript
* **Backend:** Node.js / Express
* **Database:** PostgreSQL (Relational Data Modeling)
* **Version Control:** Git & GitHub

---

## 🚀 Active Solo Enhancements & Roadmap
As part of my post-graduation development, I am independently working on the following system upgrades:
* **Database Optimization:** Refactoring complex PostgreSQL queries and indexing structures to maximize transaction speed and data retrieval efficiency.
* **Security Hardening:** Implementing robust input sanitization, secure session handling, and defensive coding parameters to mitigate common OWASP Top 10 vulnerabilities (such as SQL Injection and XSS).
* **Code Refactoring:** Modularizing backend API routes to achieve cleaner separation of concerns and improved memory management.
